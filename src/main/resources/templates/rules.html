ワンナイト人狼

ワンナイト人狼のルール説明

  1. カードの配布

    2. 6枚中4枚のカードがプレイヤーに配られ、残り2枚は墓地に起これる。

    6枚のカードには人狼が2枚、市民が2枚、占い師、怪盗がある。

  2. 夜のアクションを行う

    3. アクションは次の順で行う。

    -占い師-

     「他プレイヤー1人の役職を見る」または「選ばれなかった2つの役職を見る」のどちらかの行動を行う。

    -人狼-

     誰が人狼チームかを確認する。ただし、墓地に人狼が入っている場合はなど、人狼が1人しかないこともある。

    -怪盗-

     自身の役職と他の役職を交換することができる。交換は必須ではない。

  3. 昼のアクションを行う

    4. 誰を処刑するかを決める「議論タイム」と「投票」を行う。

  4. 勝敗の決定

    5. 勝利条件

    -人間チーム-

     人狼を処刑できたら勝利

    -人狼チーム-

     人間を処刑できたら勝利
